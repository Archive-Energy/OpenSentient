# Contradiction Synthesis

The dialectical layer of the Alethic Method.

When two positions contradict, the naive move is to pick one.
The Alethic move is to find the synthesis — the position that
holds both as partially true and explains the contradiction.

## Approach
Hegelian dialectic adapted for domain reasoning.
Thesis + antithesis -> synthesis.
The contradiction is not a problem to eliminate — it is information.

## Process
1. State both positions explicitly (thesis, antithesis)
2. Identify what each gets right
3. Identify the conditions under which each is true
4. Propose synthesis: a position that explains both
5. Identify what new inquiry the synthesis opens
6. Write synthesis as updated position node
7. Archive original positions in ## History section of node

## Output
thesis, antithesis
synthesis_position
synthesis_confidence
new_inquiry (if synthesis opens a question)
archived_positions[]

## Alethic Role
Contradictions are moments of maximum uncovering. Two truths
that can't both be true reveal a hidden third truth. This layer
finds it.
